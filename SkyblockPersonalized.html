API Features

/dungeons [user] - Returns the general dungeons stats for a player

/dungeons [user] [floor] - Returns the general dungeons stats for a player on a specifc floor

/dungeons party - Returns the general dungeons stats for your party

Commands

/sbp - opens the main menu for SkyblockPersonalized

/showconfig - Shows the config for troubleshooting purposes

/sbplink - Gives a link to the mod's website

/silverfish - Puts an image on-screen of how to solve the silverfish puzzle for x seconds (you can set how long in the /sbp menu)

/reparty - Disbands and parties everyone back that was in the party
